# Setup #
Download the preprocessed [CIFAR-10](https://drive.google.com/file/d/1AJ2aDsxO6KgP7U6zMSyXTrHZ9qcC3jP5/view?usp=sharing) and [ImageNet](https://drive.google.com/file/d/1PKxLfTnG3i8rHiTvUPfmpOvByf4vVzQW/view?usp=sharing) image feature and place it under the project folder first!

A sample code of loading the dataset and running RBF-kernel SVM training is provided in ``svm.py``, simply run
```
python svm.py
```
